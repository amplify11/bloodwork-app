# BloodWork Tracker

React + Vite app for tracking blood work biomarkers over time. Data is stored in `localStorage`.

## Develop

```
npm install
npm run dev
```

## Deploy

Pushes to `main` auto-deploy to GitHub Pages via `.github/workflows/deploy.yml`.

Enable once in repo Settings → Pages → Source: **GitHub Actions**.

Live: https://amplify11.github.io/bloodwork-app/
